# CineMax

CineMax — десктопное JavaFX-приложение для управления кинотеатром и онлайн-бронирования билетов. Система поддерживает три роли пользователей: гость, пользователь и администратор.

## Возможности

Приложение автоматизирует основные процессы кинотеатра:
- просмотр каталога фильмов с поиском по названию;
- просмотр расписания сеансов;
- бронирование и отмена билетов;
- просмотр трейлеров фильмов;
- оставление отзывов и оценок к фильмам;
- управление фильмами, сеансами и пользователями (администратор).

## Технологии

| Компонент             | Технология                              |
|-----------------------|-----------------------------------------|
| Язык                  | Java 21                                 |
| UI-фреймворк          | JavaFX 21                               |
| Система сборки        | Apache Maven                            |
| База данных           | PostgreSQL (Supabase)                   |
| Взаимодействие с БД   | Supabase REST API, java.net.http        |
| Десериализация JSON   | Gson                                    |
| Тестирование          | JUnit 5                                 |
| Паттерн               | MVC (Model-View-Controller)             |
| Хранилище постеров    | Supabase Storage                        |
